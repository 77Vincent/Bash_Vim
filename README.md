# Bash and Vim Personal settings

Personal .bash_profile and .vimrc settings on mac os x to make Bash and Vim easier to use

## What's included

* .bash_profile

* .vimrc

* color schemes

## Installation

    git clone https://github.com/77Vincent/vinvim.git
    
    cd vinvim

    ./install

    :PluginInstall

## Features

* surpertab
* autoclose
* jsBeautify
* mutilple syntax highlight
